# Accept a Pull Request (PR) on GitHub

Please follow these steps or check the GIF at the bottom to learn how toaccept a Pull Request (PR) on GitHub.

## 1. Got to the Pull Requests tab

Navigate to the **Pull Requests** tab in your GitHub repository (check screenshot below) to see a list of all the Pull Requests.

![Pull Request Tab](https://github.com/CellMigrationLab/LabConstrictor/blob/doc_source/PR_tab.png)

## 2. Select the Pull Request to review

Usually, the latest Pull Request will be at the top of the list. Click on the title of the Pull Request you want to review.

## 3. Merge the Pull Request

Click the green **Merge pull request** button and then merge by clicking **Confirm merge**.

## 4. Delete the branch (optional)

After merging, you can delete the branch associated with the Pull Request by clicking the **Delete branch** button.

![Accept a Pull Request GIF](https://github.com/CellMigrationLab/LabConstrictor/blob/doc_source/Accept_PR.gif)
